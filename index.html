Onion Shelf Life and Spoilage Detection Using Thermal Image Processing

This project leverages thermal image processing and machine learning to assess the internal quality and predict the shelf life of onions. Using a custom-built MATLAB GUI, thermal images are preprocessed, analyzed for temperature anomalies, and classified using CNNs. A regression model further predicts remaining usable days based on damage and weight loss. The system helps detect early spoilage without physical inspection, offering a practical solution for post-harvest quality management. It demonstrates high accuracy in classifying onion types and predicting degradation trends across storage durations.
